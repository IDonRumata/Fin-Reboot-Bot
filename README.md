## 🔥 Ключевые возможности

* **Smart Sales Funnel:** Встроенная воронка: "Бесплатный урок -> Оффер -> Оплата -> Обучение -> Upsell".
* **Multi-Payment Core:** Готовая интеграция с **bePaid** (СНГ/Мир), **ЕРИП** (Беларусь) и **Telegram Stars**.
* **Telegram Mini Apps (TMA):** Встроенные микро-приложения (Инвест-калькулятор и Финансовый чекап) для повышения вовлеченности.
* **AI-Mentor:** Claude API отвечает на вопросы учеников по материалам курса в режиме 24/7.
* **Automated Content Delivery:** Выдача уроков строго по расписанию (напр. раз в 24 часа) с проверкой квизов.
* **Admin Dashboard:** Полный контроль статистики, аналитика оплат и возможность массовых рассылок.

## 🛠 Технический стек

* **Backend:** Python 3.11 (aiogram 3.x)
* **Automation:** n8n (сложные цепочки вебхуков и интеграций)
* **Frontend:** Telegram Mini Apps (HTML/JS)
* **AI:** Claude API (Anthropic)

## 🚀 Архитектура воронки

1. **Enter:** Пользователь жмет `/start` -> получает Welcome-пакет и бесплатный прогревающий контент.
2. **Pay:** Интегрированная кнопка оплаты. После транзакции бот мгновенно меняет статус пользователя.
3. **Learn:** Авто-выдача уроков. Бот не выдаст следующий день, пока не пройден квиз предыдущего.
4. **Interact:** Использование Mini Apps для расчетов и AI-чата для уточнения деталей.
5. **Expand:** В конце курса бот предлагает продукт следующей ступени (Upsell).
