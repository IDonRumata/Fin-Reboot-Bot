# Финансовая перезагрузка — Telegram Bot

Telegram-бот для проведения 5-дневного инвестиционного курса.

## Технологический стек

- **Python 3.11+**, **aiogram 3.x** — асинхронный Telegram-фреймворк
- **PostgreSQL 16** — основная БД (пользователи, прогресс, платежи, контент)
- **Redis 7** — FSM (состояния) и кеширование
- **NocoDB** — Google Sheets-подобный UI для редактирования контента
- **APScheduler** — фоновые задачи (планировщик дней, напоминания)
- **Docker Compose** — деплой всего стека одной командой

## Быстрый старт

```bash
# 1. Скопируйте .env
cp .env.example .env
# Заполните BOT_TOKEN, ADMIN_IDS и другие значения

# 2. Запустите всё
docker compose up -d

# 3. Бот автоматически:
#    - Создаст таблицы в PostgreSQL
#    - Загрузит контент из data/content_blocks.csv
#    - Начнёт отвечать на команды
```

## NocoDB — редактирование контента

После запуска откройте **http://localhost:8080** — это NocoDB, подключённый к PostgreSQL.

Вы увидите все таблицы бота (`users`, `content_blocks`, `user_progress` и т.д.) в интерфейсе, похожем на Google Sheets. Можно редактировать контент прямо из браузера.

## Структура проекта

```
bot/
├── core/           # Настройки, инициализация бота
├── database/       # Модели SQLAlchemy, CRUD-репозитории
├── handlers/       # Роутеры: /start, /menu, buy, progress, day_done, keywords...
├── middlewares/    # DB-сессия, антифлуд, логирование
├── services/       # Контент-движок, bePaid, прогресс-бар
├── workers/        # Планировщик дней, напоминания (APScheduler)
└── main.py         # Точка входа
```

## Админ-команды

| Команда | Описание |
|---------|----------|
| `/admin` | Панель администратора |
| `/stats` | Статистика: пользователи, оплаты, прохождение, лиды |
| `/test_send 12345 1` | Тестовая отправка дня 1 пользователю 12345 |
| `/confirm_payment 12345` | Ручное подтверждение оплаты |
| `/sync` | Перезагрузка контента из CSV |

## Ключевые файлы

| Файл | Назначение |
|------|-----------|
| `data/content_blocks.csv` | Учебный материал (5 дней × N блоков) |
| `.env` | Токены, ключи, URL-ы |
| `docker-compose.yml` | Postgres + Redis + NocoDB + Bot |
| `bot/database/models.py` | Все модели БД |
| `bot/services/content_sender.py` | Движок отправки контента |
